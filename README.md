# Skip SDK

WIP SDK for the Skip API. Not ready for use.

## Running Tests

To run tests simply run:

```bash
npm test
```

Watch mode:

```bash
npm run test -- --watch
```

Get Coverage:

```bash
npm run test -- --coverage
```
